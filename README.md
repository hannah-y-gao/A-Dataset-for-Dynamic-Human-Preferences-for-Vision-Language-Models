# A-Dataset-for-Dynamic-Human-Preferences-for-Vision-Language-Models

This codebase is an ongoing work to create an automated data-generation pipeline for real-time human preferences. 
The full pipeline includes a VLM-powered generation and revision process, with an option for a bounding-box (BB) variation of the dataset.

This work was first introduced in the following paper and at the 2026 DataMFM workshop at CVPR: 
```bibtex
@misc{gao2026datasetdynamichumanpreferences,
      title={A Dataset for Dynamic Human Preferences for Vision Language Models}, 
      author={Hannah Gao and Dylan Hadfield-Menell and Rachel Ma},
      year={2026},
      eprint={2606.07653},
      archivePrefix={arXiv},
      primaryClass={cs.CV},
      url={https://arxiv.org/abs/2606.07653}, 
}
```

Several updates have been made to the original implementation, including enforcing similar verbs in the actionable answer choices of prompts to encourage greater image dependence.

For a sample dataset (including both regular and BB variants) generated by the original pipeline, click [here](https://huggingface.co/datasets/hanngao/visual_human_preferences).
